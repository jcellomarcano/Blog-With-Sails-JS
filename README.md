# Blog-With-Sails-JS
Simple Blog with Sails JS

# To Install:
1) clone the repository with git:

$ git clone https://github.com/ricdizio/Blog-With-Sails-JS/
  
2) run npm install to install all the packages

$ npm install 

3) execute server with sails lift

$ sails lift --port 80
